# DigitalHub


*	UI Framework - **SwiftUI**
* Architecture - **MVVM**
* Bindings - **Combine**
* Networking - **Moya**
* Database - **SwiftData**
* Resources - **SDWebImage**
* Dependencies - **Swift Package Manager** 
* Tests - **Unit Tests**
